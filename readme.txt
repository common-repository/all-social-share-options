== Description ==

All social share WordPress Plugin integrated with latest tweets, facebook/twitter/google plus social count with 3 stylish designs.

[Full Documentation](http://www.codiblog.com/2015/11/all-social-share-options-wordpress-plugins.html)

**Plugin Features**

- Social sharing support for Facebook, Google+, Twitter.
- Latest twitter tweet
- Instgram photos support
- Dynamic load
- Support shortcode
- 3 stylish designs. 
- 100% responsive. 
- Reorder Social icon order easily.
- No Image, button created using CSS3.

== Installation ==

= Installing the plugin =

1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *All Social Share Options* and click "Install now"
2. Alternatively, download the plugin and upload the contents of `all-social-share-options.zip` to your plugins directory, which usually is `/wp-content/plugins/`. Activate the plugin.

[Full Documentation](http://www.codiblog.com/2015/11/all-social-share-options-wordpress-plugins.html)
